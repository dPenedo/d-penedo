# dpenedo.com

Personal website and blog of Daniel Penedo Lázaro — built with [Astro](https://astro.build), Tailwind CSS, and TypeScript. Deployed on Netlify.

## About

I'm a full-stack developer with a focus on Python backend and data engineering. I work on production connector systems (PHP/Laravel, WooCommerce, Shopify) and build data pipelines for sociological research — web scrapers, PostgreSQL storage, and data visualization with Pandas and Plotly.

Currently based in Mar del Plata, Argentina. Returning to the Basque Country soon.

Native in Spanish and Basque (Euskera). Professional working proficiency in English and Catalan.

## Site

- **Blog**: posts on Python, Linux, data tools, and development
- **Projects**: data scrapers, web apps, research tools

## Stack

- [Astro](https://astro.build) — static site generator
- [Tailwind CSS](https://tailwindcss.com) — styling
- [TypeScript](https://www.typescriptlang.org) — type safety
- Deployed on [Netlify](https://netlify.com)

## Local development

```sh
pnpm install
```
